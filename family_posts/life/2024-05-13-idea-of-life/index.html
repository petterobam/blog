    【示例四】一些更好生活的思维方式

    # 心智

我们自己不是我们自己，你的大脑可以思考你的大脑。
- [x轴] 自己可以分为：我知道的、我不知道的
- [y轴] 别人可以分为：他知道我的、他不知道我的

所谓心智：大脑并不是主宰，你可以控制大脑的思考，可以发现【我不知道的（含别人知道、不知道我的）】，并不断积累。
- 每个人都有自己的心智，他们存储了自己的思维方式。
- 心智会使人进入一个新的思考层次，但有时也会使人更固执，而无法进步。
- 心智转型或进阶是思考自己的思考方式和方向，它的存在可以打破了一个巴掌拍不响的限制，是可以无限突破的领域。

# 学习和探索

- [学习] I know what I will know
- [探索] I don't know what I don't know

# 学习屏障

- b1: 知而不能言（学会但无法讲述）
- b2: 言而不能记（可以讲述但无法系统化记录）
- b3: 记而不能授（可以系统化记录但无法传授他人）
- If: 从上而下突破每个屏障，学习进入到一个新的境界。
- PS: 一般突破 b2 就能创造附加价值，形成二次产物。

# 一念四转

- 用于 [是|否]（二元）问题的决策
- Step1: [是] **会** 发生什么
- Step2: [是] **不会** 发送什么
- Step3: [否] **会** 发生什么
- Step4: [否] **不会** 发生什么
- Finally: Answer it

# 成熟

- 成熟不是为了走向复杂，而是为了抵达天真

# 心态调整

心情不稳定（含过于兴奋、抑郁、伤心、烦躁和无聊等）时候，请以下思考。

## Permission-First

与自己沟通，先要允许自己与你沟通。

### 允许事实
- Assign: 接受即允许，忘记即允许，抗拒即允许
- First：无惧痛苦
- Step1: 允许痛苦的存在
- Step2: 允许痛苦的伤害
- Step3: 允许痛苦的离开

### 忘记/包容某人
- Step1: 允许某人的存在
- Step2: 允许某人的影响
- Step3: 允许自己对某人当前真实的心情

### 消除坏情绪
- Step1: 允许坏情绪的产生
- Step2: 允许坏情绪的影响
- Step3: 允许坏情绪的离开

## 空心态
- Step1: 承认自己的平凡
- Step2: 允许自己的平凡
- If: 不执着抓着某些东西
- Then: 有意识的去做，才能获取更多这种东西
- Example: 空竹流水，不截水流，方得长久

## 信任信任本身
- Think: 如今的我们很难完全相信他人，甚至自己
- Becaus: 信任只有 0 和 1，没有 0-100
- Think: 难道一丝怀疑都没有吗？
- Think: 100% 相信常规逻辑从没可能？
- First: 尝试 [信任] -> [信任] 这个动作
- For all，[信任] 真的只是 [信任]
- So that，你就可以 [信任] 任何你 [信任] 的对象
- Finally，信到无须去信，不信也不是问题
- Hope: 平和自然，安然喜悦

## 真实的寿命
- [365 天] == [1 天]？
- false when，真正平和安然的时刻
- false when，真正欣喜大笑的时刻
- false when，真正感动激动的时刻
- false when，真正痛苦流涕的时刻
- false when，真正感受活着的时刻
- ...
- How: 争取活的更久
- PS More: 重复并非无意义，学会观察正向的质变
- Finally: 你的感受才是最重要的，勇于批判自己的 false，不吝肯定自己的 true。

## 游戏人生
- If: 世界就是一个游乐场
- You: 带着赤子之心，真诚的玩每个游戏
- If: 现实中的每个问题都对应着某种游戏
- Like: 不和、矛盾、冲突都是游戏规则造成
- So: 无所谓善恶好坏
- Try: 改变自己对游戏的态度
- But: 生活不是一盘必须厮杀的游戏，生活是一盘可以握手言和、尽情享受的智慧游戏。
- We: 如孩童般，得之我乐，失之再来

## 无分别心
- If: 不用自己的标准评判当下，Then: 就会顺势做出负责任的行为。
- Think: 念头如流水，截堵它会使水流升高，不要去介怀你的任何念头，没有好坏，让他如流水般畅快流去。

## 佛性
- First: 此佛非鬼神之类，造世之主，乃自觉、自察
- If: 人人皆有佛性
- Try: 去参禅拜佛乃是让佛体验自己非佛
- For: 这个至善至美的存在，不用去刻意的寻找，它一直在我们心中
- Because: 人人皆智慧具足，需时时自问且自答，体验自身的智慧

## 90%的烦恼
- ↑→: 源于你想去解决别人的问题
- Sometimes: 我们为别人承担了部分责任，试图去解决别人的问题
- Then: 容易让别人失去了解决问题的能力
- Because: 别人的问题只有他们自己才能解决，否则那就是自己的问题
- Finally: 自己的问题，哪有那么多烦恼

## 一念天堂，一念地狱
- Example: 在同样的场景可能会产生天堂和地狱的感觉
- For me: 主要看自身如何看待这件事
- Maybe: 尴尬是因为过分的揣测
- Finally: 开心不需要理由，please break tradition！

## 贪婪不会幸福
- First: 我们都有将来很想做的事
- But always: 想成功、更成功后再去做
- Always: 总认为没有准备好
- Like: 欲望永无止境
- Please: 想做就现在去做
- Example: 手机没充满电也能打电话

## 自我验证
- Assign: 自己对自己的评判会自我验证
- Like: 一直认为自己不行，潜意识会验证
- Think how: 心想事成
- Most: 决定于你想的是什么了
- A: 自认为受害者终会成为受害者
- B: 自认为可以坚持终究能够坚持？
- !B: 自认为不可以坚持终究会放弃
- Finally: 你相信的越深越容易实现

## 骆驼、狮子、婴儿
- A: 骆驼，我该，在“责任”下生活
- B: 狮子，我要，在“目标”下生活
- C: 婴儿，我是，在“生命”中生活
- For: 我和我要体现生命的纠结与执着
- Assign: 我是才是生命的真谛
- Try: 返璞归真，感受生命

## 别人都是对的
- For Self Think: 每个人的理解和认知都是对的
- So: 你要理解别人必须先腾空自己
- Try: 不带偏见，不带批判
- For that: [人人皆对] 是站在他人角度、认知和方向上思考
